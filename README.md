# The Normal Way

Simply copy these commands and paste in Termux:
```apt-get update
apt-get upgrade -y
pkg upgrade -y
pkg install python wget -y
wget https://raw.githubusercontent.com/kaal0408/LUCIFERX/main/telesetup.py
pip install telethon

```
